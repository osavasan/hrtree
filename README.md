# HR TREE

HR TREE is a sample b-tree for HR.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements or just execute setup.sh.

```bash
./setup.sh
```

## Usage

```bash
./run.sh
```

or 

```bash
python3 app.py
```

## Test

Execute test.sh to run tests.
